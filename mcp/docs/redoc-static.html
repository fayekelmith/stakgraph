/nodes

Get nodes of a specific type

query Parameters
node_type
string
Enum: "Page" "Function" "Class" "Trait" "Datamodel" "Request" "Endpoint" "UnitTest" "IntegrationTest" "E2etest"
Example: node_type=Function

Type of node to retrieve

concise
boolean
Default: false

Only include name and file in returned data

ref_ids
string
Example: ref_ids=123,456,789

Comma-separated list of reference IDs

output
string
Default: "json"
Enum: "json" "snippet"

Output format

language
string
Example: language=typescript

Filter by programming language

Responses

Local development server

http://localhost:3355/nodes
